### CodeBook for the Programming Assignment of the Week 4

WARNING: There is a dependency on the dplyr and lubridate libraries

* How does the script work?
- Create a folder
- Download the zip file and extract the contents
- Set the root of the folder as working directory
- Load data frames from the files:
  + features.txt
  + activity_labels.txt
  + X_train.txt
  + y_train.txt
  + subject_train.txt
  + X_test.txt
  + y_test.txt,
  + subject_test.txt

- Rename the column name using the data from the data frame created from features.txt
- Replace the activity integer with the string. Activities are defined below
- Merge the 2 data frames and then select the measurements columns and apply the mean function to those measurements.
- Calculate the average by measurements, activity and subject
- Save the new data set into a file called tidy_df.txt
